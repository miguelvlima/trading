## 1. Objetivo do projeto

Construir uma **plataforma de análise e decisão de trading** em modo **PAPER** (sem execução de ordens reais), que permita:

1. **Ver o mercado** — histórico e tempo real (velas, indicadores, gráfico unificado)
2. **Gerar sinais** — estratégias rule-based (RSI, MACD, SMA/EMA, Bollinger) com explicação do *porquê*
3. **Simular estratégias** — backtest histórico realista (fees, slippage, walk-forward, benchmark)
4. **Aprender com runs passados** — memória institucional (lições + recomendações entre simulações)

A visão de longo prazo: **pesquisar → sinal → simular → aprender → repetir**, com dados reais (IBKR) e eventualmente paper trading automático. **Live trading real numa outra fase.**

### Stack

- **Backend:** FastAPI + PostgreSQL + SQLAlchemy/Alembic
- **Frontend:** React + TypeScript + Vite
- **Auth:** JWT multi-utilizador
- **Deploy:** frontend Vercel, backend/DB em cloud, CI GitHub Actions

## 5. Como começar

1. Clonar o repo e ler [`development-environments-prompt-first.md`](./development-environments-prompt-first.md)
2. `npm install` → `npm run dev:all` (Docker Desktop ligado)
3. Login em http://localhost:5173 com credenciais dev
4. Explorar as 3 tabs: **Mercado**, **Sinais**, **Simulação**
5. Correr um backtest → ver **Análise crítica** → na próxima simulação ver **Lições** e **Recomendações**
6. Escolher uma frente e abrir `feature/<tema>` a partir de `develop`
